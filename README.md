## VITABLEND
VitaBlend is an interactive smoothie recipe website developed using HTML, CSS, and JavaScript. The project focuses on presenting healthy smoothie options with detailed nutritional information in a clean and user-friendly interface.

## Features
- Display of multiple smoothie recipes  
- Interactive recipe cards  
- Dynamic pop-up (modal) with detailed information  
- Nutritional details including calories and vitamins  
- Responsive and visually appealing UI design  

## Technologies Used
- HTML  
- CSS  
- JavaScript (beginner level)

## How It Works
- Recipes are stored in a JavaScript array  
- The website dynamically renders recipe cards using JavaScript  
- When a user clicks on a recipe, a modal opens showing:
  - Ingredients  
  - Nutritional information  
  - Preparation steps
